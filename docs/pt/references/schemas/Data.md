# Data Schema

| Prop   | Type      | Default | Description              |
| ------ | --------- | ------- | ------------------------ |
| src    | `string`  | N/A     | URL to image             |
| title  | `string`  | N/A     | Image title              |
| alt    | `string`  | N/A     | Placeholder text         |
| border | `boolean` | `false` | Show border around image |
| width  | `string`  | N/A     | Image width              |
